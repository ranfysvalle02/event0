# event0
# Realm App Services + HTML/JS/CSS 

# Limitation for Demo
- No resale capability available (yet)

# Demo Features
- Register/Login
- Purchase tickets
- View Tickets
- embedded Atlas Charts using "On-Demand Materialized Views" @ Event Page
- QR Scanner / Checkin for Event Admin

- Event Management for event owners
